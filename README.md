## Overview

This project was originally built with Flask and has been migrated to Django. It currently provides a minimal Django setup with a single landing page. Further features can be added on top of this foundation.

## Quick Start

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

## Project Structure

```
config/      # Django project configuration
tracker/     # Core application with views and templates
```
